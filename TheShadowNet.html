Useful commands for security experts

Networking Commands:

  • ipconfig /all - Displays detailed network configuration.
  • ping [ip/domain] - Tests connectivity to a specified address.
  • tracert [ip/domain] - Traces the route to a destination.
  • nslookup [domain] - Queries DNS records.
  • netstat -ano - Displays active network connections and listening ports.
  • route print - Shows the routing table.
  • arp -a - Displays the ARP cache, showing IP to physical address mappings.

Netsh WLAN Commands:

  • netsh wlan show networks - Displays available Wi-Fi networks.
  • netsh wlan show interfaces - Shows wireless network adapter information.
  • netsh wlan show profiles - Lists saved Wi-Fi profiles.
  • netsh wlan export profile folder=C:\ key=clear - Exports Wi-Fi profiles with passwords.
  • netsh wlan connect name="NetworkName" - Connects to a specific Wi-Fi network.
  • netsh wlan show all - Displays all wireless network information.

Other Useful CMD Commands:

  • tasklist - Lists running processes.
  • taskkill /F /IM process.exe - Forcefully terminates a process.
  • systeminfo - Displays detailed system information.
  • sfc /scannow - Scans and repairs system files.
  • chkdsk C: /f - Checks and repairs disk errors.
  • diskpart - Launches disk partitioning tool.
  • powercfg /energy - Generates an energy efficiency report.
  • driverquery - Lists installed device drivers.
  • wmic - Windows Management Instrumentation Command-line.
  • netsh advfirewall set allprofiles state off - Disables Windows Firewall (use with caution).
  • reg query HKLM /f password /t REG_SZ /s - Searches for stored passwords in registry.
  • cipher /W:C: - Securely wipes free space on drive C:.
  • tasklist /svc - Lists running processes and associated services.
  • schtasks /query /fo LIST /v - Displays detailed information about scheduled tasks.
  • wmic startup get command,caption - Lists programs that run at startup.
  • net user - Lists all user accounts on the system.
  • net localgroup administrators - Shows members of the administrators group.
  • wmic qfe list brief - Lists installed Windows updates.
  • whoami /all - Displays current user's privileges and group memberships.
  • netsh advfirewall show allprofiles - Shows Windows Firewall configuration.
  • wmic process get caption,executablepath,commandline - Lists running processes with full command lines.
  • sc query - Lists all services and their statuses.
  • reg query HKLM\Software\Microsoft\Windows\CurrentVersion\Run - Shows programs set to run at startup (Registry).
  • wevtutil qe Security /c:25 /f:text - Displays the last 25 security event log entries.
  • cipher /c - Shows encryption status of the current directory.
  • net share - Displays information about shared resources on the computer.
  • netsh interface ipv4 show subinterfaces - Shows detailed network interface information.
  • powershell "Get-WmiObject -Class Win32_Product | Select-Object Name, Version" - Lists installed software.
  • wmic os get osarchitecture - Shows if the system is 32-bit or 64-bit.
  • wmic logicaldisk get caption,description,providername - Lists disk drives and their types.
  • certutil -store -user my - Lists personal certificates for the current user.
  • gpresult /r - Shows which Group Policy settings are applied to the current user and computer.

Commands for Developers and Programmers:

  • git status - Shows the status of the current Git repository.
  • npm list - Lists installed npm packages.
  • python --version - Displays the installed Python version.
  • javac -version - Shows the Java compiler version.
  • dotnet --info - Displays information about the installed .NET SDK.
  • docker ps - Lists running Docker containers.
  • kubectl get pods - Lists Kubernetes pods in the current context.

Commands for Cybersecurity Experts:

  • netstat -nao | findstr LISTENING - Shows all listening ports and their associated PIDs.
  • tasklist /m - Lists all running processes and their associated DLLs.
  • wmic process where processid="PID" get commandline - Shows the command line of a specific process.
  • reg query HKLM\SOFTWARE\Microsoft\Windows\CurrentVersion\Run - Lists autorun programs in the registry.
  • schtasks /query /fo LIST /v - Lists all scheduled tasks with details.
  • wevtutil qe Security /c:100 /f:text - Displays the last 100 security event log entries.

Commands for Checking if Machine is Hacked:

  • netstat -ano - Look for suspicious outbound connections.
  • tasklist /v - Check for unfamiliar processes.
  • wmic startup list full - Examine startup programs for anything suspicious.
  • wmic process list full - Review all running processes in detail.
  • dir %SystemRoot%\prefetch - Check recently run programs.
  • wevtutil qe Security /c:50 /f:text - Review recent security events.
  • schtasks /query /fo LIST /v - Check for suspicious scheduled tasks.
